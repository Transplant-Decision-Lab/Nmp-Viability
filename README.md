# NMP Graft Viability Score

Pre-specified pilot score for assessing liver graft viability during **normothermic machine perfusion (NMP)**, primarily designed for the OrganOx Metra device.

This is a **research tool** only. It is not a medical device and is not validated for standalone clinical decision-making.

---

## Features

- **Original Pilot Score (0–4)**  
  Literature-anchored core criteria:
  - Lactate ≤ 2.5 mmol/L at \~4 h
  - ALT < 6000 U/L at \~2 h
  - Bile production present
  - Lactate falling in the first 2 hours

- **Extended Pilot Score (0–9)**  
  Adds supporting criteria drawn from VITTAL, Cambridge, and Groningen frameworks:
  - Hepatic artery flow ≥ 150 mL/min
  - Portal vein flow ≥ 500 mL/min
  - Glucose metabolism
  - Bile pH ≥ 7.45
  - Δ pH (bile − perfusate) ≥ 0.10

- Clear evidence section with direct links to key papers
- Works offline once installed
- Installable as an app on both iPhone and Android

## Intended Use

This tool is intended for:
- Research and hypothesis generation
- Teaching and case discussion
- Methodological development of NMP viability assessment

It is **not** intended for:
- Standalone clinical decision-making
- Replacing multidisciplinary team assessment
- Regulatory or commercial use

---

## Relationship to Published Criteria

| Framework              | Alignment |
|------------------------|---------|
| **VITTAL / Birmingham** | Core score closely follows mandatory lactate + supporting criteria |
| **Cambridge**           | Shares ALT threshold and emphasis on lactate trajectory + bile chemistry |
| **Groningen**           | Bile pH and Δ pH criteria included in the Extended score |
